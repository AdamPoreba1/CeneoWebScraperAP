Description

The Ceneo WebScraper is a Python-based project that extracts, processes, and visualizes data from ceneo website. It integrates several powerful libraries to create a seamless data pipeline from web scraping to data presentation.

Key Features

  • Data Extraction: Utilizes the requests library to retrieve HTML content from target websites.
  • Data Parsing: Uses BeautifulSoup for navigating and extracting specific data points from HTML and XML documents.
  • Data Processing: Employs pandas to clean and transform the scraped data for analysis.
  • Data Visualization: Integrates matplotlib and seaborn to generate insightful visualizations.
  • Web Interface: Implements a web-based dashboard using Flask, providing a user-friendly interface to interact with the data and visualizations.

Libraries Used

  • requests: For fetching web page content.
  • BeautifulSoup: For parsing HTML and XML documents.
  • pandas: For data manipulation and analysis.
  • matplotlib For creating visualizations.
  • Flask: For developing the web interface and dashboard.
